24 ## The Utilities
25 
26 - <a href="chgres_cube/index.html">chgres_cube</a> - Creates cold
27  start initial conditions for FV3 model runs.
28 
29 - <a href="cpld_gridgen/index.html">cpld_gridgen</a> - Utility to
30  create the Fix and IC files for the S2SW and S2S applications
31 
32 - <a href="ocnice_prep/index.html">ocnice_prep</a> - Utility to
33  create the warmstart files for OCN and ICE from component restart
34  files at 1/4 deg tripole resolution
35 
36 - <a href="emcsfc_ice_blend/index.html">emcsfc_ice_blend</a> - Blends
37  National Ice Center sea ice cover and EMC sea ice concentration data
38  to create a global sea ice analysis used to update the GFS once per
39  day.
40 
41 - <a href="emcsfc_snow2mdl/index.html">emcsfc_snow2mdl</a> - Blends
42  National Ice Center snow cover and Air Force snow depth data to
43  create a global depth analysis used to update the GFS snow field
44  once per day.
45 
46 - fre-nctools - Tools to remap data; and to create the geo-reference
47  fields (latitude, longitude, etc.) for an FV3 grid.
48 
49 - <a href="fvcom_tools/index.html">fvcom_tools</a> - Replaces lake
50  surface and lake ice temperature along with aerial ice concentration
51  generated from the Great Lakes Operational Forecast System (GLOFS)
52  in an FV3 surface restart file.
53 
54 - <a href="gblevents/index.html">gblevents</a> -
55  Prepares observational prepbufr reports for subsequent quality
56  control and analysis programs.
57 
58 - <a href="global_cycle/index.html">global_cycle</a> -
59  Updates the GFS surface conditions using external snow and sea ice
60  analyses. Updates monthly climatological fields such as plant
61  greenness fraction and albedo. Runs as part of the GFS and GDAS
62  cycles.
63 
64 - <a href="grid_tools/index.html">grid_tools</a> -
65  Utilities to filter topography, to create regional extended Schmidt
66  gnomonic grids, and to compute the equivalent global resolution of a
67  regional grid.
68 
69 - <a href="lsm_routines/index.html">lsm_routines</a> - Land surface
70  model-specific routines that are utilised elsewhere within UFS_UTILS.
71  Currently, contains the routines required by global_cycle to
72  perform data assimilation updates to land model states
73 
74 - <a href="ocean_merge/index.html">ocean_merge</a> - Utility to
75  determine the water mask by merging a lake mask with a mapped
76  ocean mask from MOM6.
77 
78 - <a href="orog_mask_tools/index.html">orog_mask_tools</a> - Utilities
79  to create land mask, terrain and gravity wave drag fields; set lake
80  fraction and depth; creates an inland land mask.
81 
82 - <a href="sfc_climo_gen/index.html">sfc_climo_gen</a> - Creates
83  surface climatological fields, such as vegetation type and albedo,
84  for an FV3 grid.
85 
86 - <a href="vcoord_gen/index.html">vcoord_gen</a> - Generates hybrid
87  coordinate parameters from fields such as surface pressure, model
88  top and the number of vertical levels. Outputs the 'ak' and 'bk'
89  parameters used by the forecast model to define the hybrid levels.
90 
91 - <a href="weight_gen/index.html">weight_gen</a> - Utility to
92  create gaussian grid ESMF 'scrip' files for use in creating
93  ESMF interpolation weight files.
